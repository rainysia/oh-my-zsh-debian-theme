debian theme for oh-my-zsh
=========================
# oh-my-zsh-debian-theme
It's for debian user use oh-my-zsh in root/user mode 

Usage
---------------------
### Details.
        1.  copy it into your ~/.oh-my-zsh/themes
        2.  edit your ~/.zshrc and change the value ZSH_THEME="debian"
